Core Library (``qgis_macros``)
==============================

The core library provides the macro recording and playback engine.
It can be used as a standalone dependency by any plugin using `qgis-plugin-dev-tools`_ ``runtime_requires``.


**Package:** ``macro-qgis-core`` (PyPI) / ``qgis_macros`` (import)
